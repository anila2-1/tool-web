In today's interconnected world, billions of digital devices communicate every second. From browsing the internet to sending emails, streaming videos, or conducting financial transactions, every online activity depends on one critical element: an **IP address**.

<h2 className="text-2xl font-semibold text-indigo-600">What is an IP Address?</h2>

An IP address (Internet Protocol Address) identifies all devices connected to a computer network that supports Internet Protocol. Think of it as the "digital address" of your device. Just as your home has a postal address, your smartphone, laptop, or server needs an IP address to send and receive information.

<h2 className="text-2xl font-semibold text-indigo-600">Types of IP Addresses</h2>

There are two primary types of IP addresses in use today:
</div>

<div className="bg-blue-50 border-l-4 border-blue-500 p-6 rounded-r-xl mb-6 hover:shadow-md transition-shadow">
  <div className="text-2xl font-semibold text-blue-700 mb-4 flex items-center gap-2">
    🌐 IPv4 (Internet Protocol Version 4)
  </div>
  <ul className="list-disc list-inside space-y-2 text-gray-700">
    <li><strong>Format:</strong> <code className="bg-white px-2 py-1 rounded border border-gray-300 font-mono text-sm">192.168.1.1</code></li>
    <li><strong>Uses:</strong> <span className="font-semibold">32-bit</span> addressing system</li>
    <li>Most widely used, but limited to ~4.3 billion unique addresses</li>
  </ul>
</div>

<div className="bg-purple-50 border-l-4 border-purple-500 p-6 rounded-r-xl mb-6 hover:shadow-md transition-shadow">
  <div className="text-2xl font-semibold text-purple-700 mb-4 flex items-center gap-2">
    🌍 IPv6 (Internet Protocol Version 6)
  </div>
  <ul className="list-disc list-inside space-y-2 text-gray-700">
    <li><strong>Format:</strong> <code className="bg-white px-2 py-1 rounded border border-gray-300 font-mono text-sm">2001:0db8:85a3::8a2e:0370:7334</code></li>
    <li><strong>Uses:</strong> <span className="font-semibold">128-bit</span> addressing system</li>
    <li>Provides over 340 undecillion unique addresses — future-proof for IoT and beyond</li>
  </ul>
</div>

<h2 className="text-2xl font-semibold text-indigo-600">How IP Addresses Identify Devices</h2>

Every time you connect to the internet, your device is assigned an IP address. This address serves two main functions:
</div>

<ul className="space-y-3 mb-6">
  <li className="flex items-start gap-3 p-3 rounded-xl bg-gradient-to-r from-gray-50 to-white border border-gray-200 shadow-sm hover:shadow-md transition-all duration-300">
    <span className="flex-shrink-0 text-lg">🆔</span>
    <span className="text-gray-700">
      <strong className="text-indigo-600">Identification:</strong> Uniquely identifies your device within a network so data reaches the correct device.
    </span>
  </li>
  <li className="flex items-start gap-3 p-3 rounded-xl bg-gradient-to-r from-gray-50 to-white border border-gray-200 shadow-sm hover:shadow-md transition-all duration-300">
    <span className="flex-shrink-0 text-lg">📍</span>
    <span className="text-gray-700">
      <strong className="text-indigo-600">Location Addressing:</strong> Provides information about the device's location in the network hierarchy.
    </span>
  </li>
</ul>

<div className="text-gray-700 leading-relaxed space-y-4 my-6">
For example, when you type a website URL, your request is sent from your IP address to the website's server. The server then knows where to send the response, much like sending a letter back to the sender's address.

<h2 className="text-2xl font-semibold text-indigo-600">IP Address Forms</h2>
</div>

<ul className="space-y-3 mb-6">
  <li className="flex items-start gap-3 p-3 rounded-xl bg-gradient-to-r from-gray-50 to-white border border-gray-200 shadow-sm hover:shadow-md transition-all duration-300">
    <span className="flex-shrink-0 text-lg">🌐</span>
    <span className="text-gray-700">
      <strong className="text-indigo-600">Public IP:</strong> Assigned by your ISP to identify your network on the internet.
    </span>
  </li>
  <li className="flex items-start gap-3 p-3 rounded-xl bg-gradient-to-r from-gray-50 to-white border border-gray-200 shadow-sm hover:shadow-md transition-all duration-300">
    <span className="flex-shrink-0 text-lg">🏠</span>
    <span className="text-gray-700">
      <strong className="text-indigo-600">Private IP:</strong> Used within a local network to identify devices like phones, printers, and smart TVs.
    </span>
  </li>
  <li className="flex items-start gap-3 p-3 rounded-xl bg-gradient-to-r from-gray-50 to-white border border-gray-200 shadow-sm hover:shadow-md transition-all duration-300">
    <span className="flex-shrink-0 text-lg">📌</span>
    <span className="text-gray-700">
      <strong className="text-indigo-600">Static IP:</strong> Permanently assigned, does not change, often used by servers or businesses.
    </span>
  </li>
  <li className="flex items-start gap-3 p-3 rounded-xl bg-gradient-to-r from-gray-50 to-white border border-gray-200 shadow-sm hover:shadow-md transition-all duration-300">
    <span className="flex-shrink-0 text-lg">🔄</span>
    <span className="text-gray-700">
      <strong className="text-indigo-600">Dynamic IP:</strong> Temporarily assigned by ISPs, common for regular internet users.
    </span>
  </li>
</ul>
