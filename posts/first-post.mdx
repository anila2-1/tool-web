<div className="prose max-w-none">
  {/* Main Heading */}
  <h1 className="text-3xl font-bold my-6">What is an IP Address?</h1>
  
  {/* Remove the p tag wrapper - let MDX handle paragraphs */}
  An <strong>IP Address</strong> (Internet Protocol Address) is a unique identifier for a device on a network.  
  It allows devices to communicate with each other over the internet or a local network.


  {/* Types Section */}
  <h2 className="text-2xl font-bold my-6 pb-2">Types of IP Addresses</h2>
  
  <h3 className="text-xl font-semibold my-4 text-blue-600">IPv4 (Internet Protocol Version 4)</h3>
  <ul className="list-disc pl-6 mb-6">
    <li className="mb-2">Format: <code className="bg-gray-100 px-1.5 py-0.5 rounded">192.168.1.1</code></li>
    <li className="mb-2">Uses <strong>32-bit</strong> numbers</li>
    <li>Most common but limited in unique addresses</li>
  </ul>

  <h3 className="text-xl font-semibold my-4 text-blue-600">IPv6 (Internet Protocol Version 6)</h3>
  <ul className="list-disc pl-6 mb-6">
    <li className="mb-2">Format: <code className="bg-gray-100 px-1.5 py-0.5 rounded">2001:0db8:85a3:0000:0000:8a2e:0370:7334</code></li>
    <li className="mb-2">Uses <strong>128-bit</strong> numbers</li>
    <li>Provides a huge number of unique addresses</li>
  </ul>

  <h2 className="text-2xl font-bold my-6 pb-2">Example IP Addresses</h2>
