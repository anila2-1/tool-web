<div className="prose max-w-none">
  {/* Main Heading */}
  <h1 className="text-3xl font-bold my-6">What is JSON?</h1>
  
  <div className="text-gray-700 mb-6">
    <strong>JSON</strong> (JavaScript Object Notation) is a lightweight data-interchange format that is:
  </div>

  <ul className="list-disc pl-6 mb-6">
    <li className="mb-2">Easy for humans to read and write</li>
    <li className="mb-2">Simple for machines to parse and generate</li>
    <li>Language-independent (works across all programming languages)</li>
  </ul>

  {/* Syntax Section */}
  <h2 className="text-2xl font-bold my-6 pb-2">JSON Syntax Rules</h2>
  
  <div className="bg-gray-50 p-4 rounded-lg mb-6">
    <ul className="list-disc pl-6 space-y-2">
      <li>Data is represented as key-value pairs: <code className="bg-gray-100 px-1.5 py-0.5 rounded">"key": "value"</code></li>
      <li>Data is separated by commas</li>
      <li>Curly braces <code>{}</code> hold objects</li>
      <li>Square brackets <code>[]</code> hold arrays</li>
      <li>Strings must use double quotes <code>""</code></li>
    </ul>
  </div>

  {/* Data Types Section */}
  <h2 className="text-2xl font-bold my-6 pb-2">JSON Data Types</h2>

      </thead>
      <tbody>
        <tr>
          <td className="py-2 px-4 ">String</td>

          <td className="py-2 px-4 "><code>{"\"age\": 30"}</code></td>

          <td className="py-2 px-4 ">Array</td>
